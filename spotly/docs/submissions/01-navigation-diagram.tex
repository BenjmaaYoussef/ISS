\textbf{Reading the diagram.}
Each rectangle represents a user interface identified by its page ID and title.
Arrows represent navigation paths between interfaces, triggered by user actions
such as button clicks, form submissions, or automatic redirects from the router
guard layer.
Interfaces are grouped into five access areas: \textbf{Public} (open to all
visitors without authentication), \textbf{Booking Flow} (multi-step reservation
journey for authenticated clients), \textbf{Client Area} (personal dashboard and
reservation management), \textbf{Admin Area} (full venue management for venue
owners), and \textbf{Staff Area} (live floor operations for assigned staff
members). Dialogs (D1--D10) are modal overlays launched from within their host
page and do not constitute standalone routes.
